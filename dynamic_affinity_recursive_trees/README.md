# Recursive trees grown from dynamic building sequences

This folder contains the LaTeX manuscript, bibliography and reproducible
Monte Carlo experiments for preferential recursive trees with dynamic
affinities (age index \(\Lambda\in[0,2]\)).

## Contents

- `main.tex`: scientific manuscript (theory + numerical illustrations)
- `references.bib`: BibTeX references
- `python/simulate_dynamic_trees.py`: main simulation pipeline (recommended)
- `R/simulate_dynamic_trees.R`: base-R smoke-test mirror
- `requirements.txt`: Python dependencies
- `results/`: CSV summaries and PDF/PNG figures generated by the simulations
- `source_main.pdf`: original source PDF used as reference

## Compile the manuscript

From this folder:

```sh
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

## Run the simulations

```sh
python3 -m pip install -r requirements.txt
python3 python/simulate_dynamic_trees.py
```

If `Rscript` is available:

```sh
Rscript R/simulate_dynamic_trees.R
```

The Python script writes:

- `results/depth_moments_summary.csv`
- `results/clt_summary.csv`
- `results/mean_variance_check.pdf`
- `results/lln_convergence.pdf`
- `results/v_lambda_curve.pdf`
- `results/clt_{young,uniform,old}.pdf`
- `results/parent_density_{young,uniform,old}.pdf`

(and PNG copies of each figure).

## Model summary

When node \(n\) arrives, it attaches to parent \(k\in\{1,\ldots,n-1\}\) with
probability

\[
\mathbb{P}(A_{n,k})=\frac{\Lambda n+2(1-\Lambda)k}{n(n-1)}.
\]

Special cases: young-age \(\Lambda=0\), uniform \(\Lambda=1\), old-age
\(\Lambda=2\). The depth \(D_n\) satisfies

\[
\frac{D_n}{\ln n}\to\frac{2}{1+\Lambda}
\quad\text{in probability},\qquad
\frac{D_n-\mathbb{E}[D_n]}{\sqrt{\ln n}}\to\mathcal{N}(0,V_\Lambda),
\]

with \(V_\Lambda=(2+8\Lambda-2\Lambda^2)/(\Lambda+1)^3\).
